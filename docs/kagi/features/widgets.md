# Widgets

## Answer Box

The answer box widget provides quick results for math computations, science queries, and general facts. It's powered by Wolfram|Alpha.

#### Example Queries
- `1+23` -> `= 24`
- `integral of x^2` -> `= x^3/3`
- `age of obama` -> `= 64 years`
- `electrons in carbon atom` -> `= 6 electrons`

## Blast from the Past

This widget sources results from the Internet Archive to show you results as they would have been back in the day.

#### Example Queries
- `windows xp`

## Computer

The computer widget offers a wide variety of calculations and useful features, such as:
- Currency conversion
- Crypto conversion
- Math
- Timezone conversion
- UUID generation
- MD5 of query
- URL encode/decode of query
- Base64 encode/decode of query
- Hex/octal/decimal conversions

#### Example Queries
- `generate uuid`
- `45ft in m`
- `1usd to eur`
- `123 in hex`

## Clock

The clock widget provides time results, such as current time in your locale or a specific timezone. Conversion queries can also be given to show the time in another location based on a given time and location.

#### Example Queries
- `current time`
- `time in amsterdam`
- `10pm boston in amsterdam`
- `9am sf in chicago`

## Color Picker

This widget presents a color picker, and if a color is specified in the query, will have that color pre-selected. 

#### Example Queries
- `#afafaf`
- `rgb(0,0,1)`
- `hsl(100 10% 10%)`

## Dictionary

The dictionary widget displays word definitions and pronunciation, powered by Kagi Translate.

#### Example Queries
- `elated`
- `chronic definition`
- `define amazing`

## Discussions

This widget surfaces relevant discussions on forums.

## Images

This widget shows relevant images inline to the results page.

## Inline Maps

This widget provides quick access to map results without having to leave the main results page.

#### Example Queries
- `coffee near me`
- `sushi near seattle`

You can also search for local restaurant names, for example `veggie galaxy` if you're in Boston.

## Interesting Finds

This widget shows in the search result column to surface some hopefully interesting, less known content.

#### Example Queries
- `steve jobs`
- `floppy disk`

## Listicles

This widget collects all "listicle" style results into one section.

## News

This widget collects news stories relevant to the query.

## Public Records

This widget shows any public records found for the given query.

## Quick Peek

This widget shows common questions and answers related to the query.

information about the package and its current status.

| Provider | Link to track | Inline tracking info |
|----------|:-------------:|:--------------------:|
| USPS     |             x |                    x |
| UPS      |             x |                    x |
| FedEx    |             x |                    x |

| Haypost  |             x |                      |
| Israel Post |          x |                      |
| Korea Post |           x |                      |
| Post Aruba |           x |                      |
| Post Nord |            x |                      |

## Stock Widget

This widget displays prices, charts, and other info for a given stock ticker query.

#### Example Queries
- `DOW`
- `google stock`
- `NVDA`
- `nvidia stock`

## Stopwatch/Timer

This widget displays a basic stopwatch/time combo. The stopwatch is capable of splits, and the timer can play sound when it ends. These widgets do not work when JS is disabled.

#### Example Queries
- `stopwatch`
- `timer`
- `5 minute timer`
- `3 min timer`

## Unix Time

This widget can convert from Unix epoch to local/UTC time.

#### Example Queries
- `1711428831 unix time`
- `10 unix`
- `unix 1716428832`

## Videos

This widget shows relevant videos inline to the results page.

## Weather

This widget shows the current and forecasted weather for the day and week. You can specify a location or use your current location. Currently this is powered by Open-Meteo.

#### Example Queries
- `weather london`
- `weather`
- `weather in sf`

## What's my IP?

This widget displays your current IP address and geoIP information. This is not stored and uses an offline lookup table to display data.

#### Example Queries
- `my ip`
- `whats my ip address`

## Wikipedia

This is the most common widget to see, it will show up with a small summary and infobox whenever a related Wikipedia article is found for your query.

#### Example Queries
- `chicago`
- `steve wozniak`
