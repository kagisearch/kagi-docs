# Setting Kagi as Default on Firefox (iPhone/iPad)

1. Tap the menu button, then **Settings** > **Search** > **Add Search Engine**.
1. Enter the following:
	- **Title**: `Kagi`
	- **URL**: `https://kagi.com/search?q=%s`
1. Tap **Save**.
1. Under **Default Search Engine**, select **Kagi**.
